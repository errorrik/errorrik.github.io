1.1. namespace

UI组件库归属于一个namespace：esui下。

1.2. 控件对象的初始化

控件对象可以通过new或通过工厂方法进行初始化。

控件构造函数只允许传入一个Object类型的参数：option，便于扩展。

1.3. 渲染页面上存在的dom元素：控件主元素

页面上当前需要渲染的元素，我们称为“控件主元素”。

控件主元素应以特殊的attribute（如“ui”）进行标识，attribute的值应遵循"name:value[;name:value]"的语法。

attribute中必须包含控件类型type属性，建议包含id属性。

通过构造函数传入的控件主元素，option用main字段。

1.4. render与appendTo

控件应提供render方法，该方法不接受任何参数。控件主元素通过构造函数参数option的“main”成员传递。

控件可以提供appendTo方法，该方法接受一个HTMLElement类型的参数做为控件主元素的父元素。其行为是将append到这个HTMLElement中，然后执行render方法的动作。控件主元素的创建通过是否在构造函数中传入控件主元素判断。
